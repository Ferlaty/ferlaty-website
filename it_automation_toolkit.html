Description:


Automation Toolkit is a small, user-friendly IT toolkit designed for:
- Viewing detailed system information
- Analysing log files
- Generating secure passwords
- Scanning your LAN (explicitly user-initiated)
-Verifying, if a port is open on the system

Scanning
 - This network scanner uses raw ICMP for maximum speed. May trigger antivirus warnings (Defender/SmartScreen).

Download Options
 - Portable (.zip)
- Installer (.exe)

Security & Transparency
 - Scans are user-initiated only. No automatic actions.
- VirusTotal reports are included for all builds.
- No files are written or modified unless you explicitly choose a tool.
